# Task 1: Iris Dataset Exploratory Data Analysis (EDA)

## Overview
This project performs **Exploratory Data Analysis (EDA)** on the classic **Iris dataset**.  
The goal is to explore the dataset, visualize patterns, and gain insights about the relationships between features, which can help in future classification tasks.

## Dataset
The Iris dataset contains **150 samples of iris flowers** with 4 features:

- Sepal Length (cm)  
- Sepal Width (cm)  
- Petal Length (cm)  
- Petal Width (cm)  

The target variable is **species**, which includes three classes:

- Setosa  
- Versicolor  
- Virginica  

## Key Features of this Project
- Data cleaning and preprocessing  
- Summary statistics of all features  
- Visualization of data distributions using histograms and boxplots  
- Pair plots and correlation analysis  
- Insights and patterns for classification  

## Tools & Libraries
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
